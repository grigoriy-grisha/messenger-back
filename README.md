

## Установка

1. `npm install` или `yarn`
2. Создать `.env` файл, с помощью команды `cp .env.example .env`.
3. Установить MongoDB если еще не установлен и запустить.
4. Выполнить команду `yarn start` или `npm start`.



**Stack:**

- NodeJS
- TypeScript
- Express
- Mongoose
- Multer
- Nodemailer
- Socket.io
- JWT
- Cloudinary
